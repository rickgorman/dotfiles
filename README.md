# dotfiles

## Installation

First, head over to [thoughtbot's dotfiles](https://github.com/thoughtbot/dotfiles) and install those. Then:

```
cd ~
git clone https://github.com/rickgorman/dotfiles.git dotfiles-local
cd dotfiles-local
./setup.sh
```
